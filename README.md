# Custom WooCommerce Invoice Generator (CIG)

**Version:** 4.0.0  
**Requires PHP:** 7.4+  
**Requires WooCommerce:** 5.0+  
**HPOS Compatible:** Yes  

Custom WooCommerce Invoice Generator (CIG) рЃљрЃарЃўрЃА рЃЏрЃФрЃџрЃљрЃЋрЃарЃў рЃЊрЃљ рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў WordPress рЃърЃџрЃљрЃњрЃўрЃюрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃўрЃюрЃЋрЃЮрЃўрЃАрЃћрЃЉрЃўрЃА, рЃЏрЃљрЃарЃљрЃњрЃћрЃЉрЃўрЃА рЃарЃћрЃќрЃћрЃарЃЋрЃљрЃфрЃўрЃўрЃАрЃљ рЃЊрЃљ рЃцрЃўрЃюрЃљрЃюрЃАрЃБрЃарЃў рЃљрЃдрЃарЃўрЃфрЃ«рЃЋрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃўрЃќрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃърЃџрЃљрЃњрЃўрЃюрЃў рЃўрЃДрЃћрЃюрЃћрЃЉрЃА **рЃАрЃљрЃЎрЃБрЃЌрЃљрЃа рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃфрЃ«рЃарЃўрЃџрЃћрЃЉрЃА (Custom Database Tables)** рЃЏрЃљрЃдрЃљрЃџрЃў рЃгрЃљрЃарЃЏрЃљрЃЊрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЊрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃАрЃарЃБрЃџ рЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃўрЃќрЃљрЃфрЃўрЃљрЃА WooCommerce-рЃўрЃА рЃЏрЃљрЃарЃљрЃњрЃћрЃЉрЃЌрЃљрЃю.

## ­Ъџђ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў

### 1. рЃўрЃюрЃЋрЃЮрЃўрЃАрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ
рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃЏрЃ«рЃљрЃарЃА рЃБрЃГрЃћрЃарЃА рЃўрЃюрЃЋрЃЮрЃўрЃАрЃўрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃАрЃљрЃАрЃўрЃфрЃЮрЃфрЃ«рЃџрЃЮ рЃфрЃўрЃЎрЃџрЃА:
* **Fictive (рЃцрЃўрЃЦрЃбрЃўрЃБрЃарЃў):** рЃўрЃюрЃЋрЃЮрЃўрЃАрЃў рЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃа рЃљрЃЎрЃџрЃЊрЃћрЃЉрЃљ рЃЏрЃљрЃарЃљрЃњрЃА рЃЊрЃљ рЃљрЃа рЃўрЃАрЃљрЃ«рЃћрЃЉрЃљ рЃерЃћрЃЏрЃЮрЃАрЃљрЃЋрЃџрЃћрЃЉрЃерЃў.
* **Reserved (рЃЊрЃљрЃарЃћрЃќрЃћрЃарЃЋрЃћрЃЉрЃБрЃџрЃў):** рЃърЃарЃЮрЃЊрЃБрЃЦрЃфрЃўрЃљ рЃЊрЃљрЃ»рЃљрЃЋрЃерЃюрЃўрЃџрЃўрЃљ рЃЎрЃџрЃўрЃћрЃюрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА (рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃўрЃЌрЃЋрЃџрЃўрЃА "Reserved Stock"-рЃА), рЃЏрЃљрЃњрЃарЃљрЃЏ WooCommerce-рЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃа рЃЏрЃљрЃарЃљрЃњрЃА рЃ»рЃћрЃа рЃљрЃа рЃљрЃЎрЃџрЃЊрЃћрЃЉрЃљ.
* **Sold (рЃњрЃљрЃДрЃўрЃЊрЃБрЃџрЃў):** рЃбрЃарЃљрЃюрЃќрЃљрЃЦрЃфрЃўрЃљ рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃўрЃљ, рЃЏрЃљрЃарЃљрЃњрЃў рЃЮрЃцрЃўрЃфрЃўрЃљрЃџрЃБрЃарЃљрЃЊ рЃЕрЃљрЃЏрЃЮрЃўрЃГрЃарЃљ WooCommerce-рЃўрЃЊрЃљрЃю рЃЊрЃљ рЃЌрЃљрЃюрЃ«рЃљ рЃљрЃўрЃАрЃљрЃ«рЃљ рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃљрЃерЃў.
* **Date Logic:** рЃцрЃўрЃЦрЃбрЃўрЃБрЃарЃўрЃЊрЃљрЃю рЃљрЃЦрЃбрЃўрЃБрЃа рЃАрЃбрЃљрЃбрЃБрЃАрЃерЃў рЃњрЃљрЃЊрЃљрЃАрЃЋрЃџрЃўрЃАрЃљрЃА, рЃњрЃљрЃДрЃўрЃЊрЃЋрЃўрЃА рЃЌрЃљрЃарЃўрЃдрЃў (`sale_date`) рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃњрЃљрЃюрЃљрЃ«рЃџрЃЊрЃћрЃЉрЃљ рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃЊрЃарЃЮрЃўрЃЌ.

### 2. рЃЏрЃљрЃарЃљрЃњрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў (Advanced Stock Management)
* **Real-time Sync:** WooCommerce-рЃўрЃА рЃАрЃбрЃЮрЃЎрЃћрЃЉрЃўрЃА рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў.
* **Prevention:** рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃљрЃа рЃњрЃљрЃњрЃљрЃДрЃўрЃЊрЃўрЃюрЃћрЃЉрЃЌ рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃА, рЃЌрЃБ рЃўрЃА рЃАрЃбрЃЮрЃЎрЃерЃў рЃљрЃа рЃљрЃарЃўрЃА.
* **Duplicate Prevention:** рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃћрЃЉрЃБрЃџрЃў SQL рЃарЃћрЃърЃЮрЃарЃбрЃћрЃЉрЃў (`GROUP BY` рЃџрЃЮрЃњрЃўрЃЎрЃўрЃЌ) рЃАрЃбрЃЮрЃЎрЃћрЃЉрЃўрЃА рЃќрЃБрЃАрЃбрЃў рЃљрЃдрЃарЃўрЃфрЃ«рЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА.

### 3. рЃарЃЮрЃџрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ
* **Admin Portal:** рЃАрЃарЃБрЃџрЃў рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃДрЃЋрЃћрЃџрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃќрЃћ.
* **Accountant View:** рЃерЃћрЃќрЃдрЃБрЃЊрЃБрЃџрЃў рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃцрЃўрЃюрЃљрЃюрЃАрЃБрЃа рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃАрЃљ рЃЊрЃљ рЃўрЃюрЃЋрЃЮрЃўрЃАрЃћрЃЉрЃќрЃћ.
* **Frontend Dashboard:** рЃАрЃърЃћрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА `wp-admin`-рЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ.

### 4. рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў
* **Custom Tables:** рЃўрЃДрЃћрЃюрЃћрЃЉрЃА `wp_cig_invoices`, `wp_cig_invoice_items`, `wp_cig_payments` рЃЊрЃљ `wp_cig_customers` рЃфрЃ«рЃарЃўрЃџрЃћрЃЉрЃА, рЃарЃљрЃф рЃАрЃљрЃњрЃарЃФрЃюрЃЮрЃЉрЃџрЃљрЃЊ рЃќрЃарЃЊрЃўрЃА рЃАрЃўрЃАрЃгрЃарЃљрЃцрЃћрЃА `wp_postmeta`-рЃАрЃЌрЃљрЃю рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃЌ.
* **AJAX Driven:** рЃфрЃ«рЃарЃўрЃџрЃћрЃЉрЃў, рЃФрЃћрЃЉрЃюрЃљ рЃЊрЃљ рЃцрЃўрЃџрЃбрЃарЃљрЃфрЃўрЃљ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА AJAX-рЃќрЃћ рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃњрЃљрЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ.
* **Frontend Assets:** Dashicons-рЃўрЃА рЃЊрЃљ рЃАрЃбрЃўрЃџрЃћрЃЉрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ рЃљрЃарЃљ-рЃљрЃЊрЃЏрЃўрЃюрЃў рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

## ­ЪЏа рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ

1.  рЃљрЃбрЃЋрЃўрЃарЃЌрЃћрЃЌ рЃърЃџрЃљрЃњрЃўрЃюрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃў `/wp-content/plugins/custom-woocommerce-invoice-generator` рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў.
2.  рЃњрЃљрЃљрЃЦрЃбрЃўрЃБрЃарЃћрЃЌ рЃърЃџрЃљрЃњрЃўрЃюрЃў WordPress-рЃўрЃА "Plugins" рЃЏрЃћрЃюрЃўрЃБрЃЊрЃљрЃю.
3.  рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ **WooCommerce** рЃљрЃЦрЃбрЃўрЃБрЃарЃўрЃљ.
4.  рЃљрЃЦрЃбрЃўрЃЋрЃљрЃфрЃўрЃўрЃАрЃљрЃА рЃърЃџрЃљрЃњрЃўрЃюрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ рЃфрЃ«рЃарЃўрЃџрЃћрЃЉрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљрЃерЃў (`dbDelta`-рЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ).

## ­Ъњ╗ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ (Shortcodes)

рЃърЃџрЃљрЃњрЃўрЃюрЃў рЃњрЃЋрЃЌрЃљрЃЋрЃљрЃќрЃЮрЃЉрЃА рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃерЃЮрЃарЃЌрЃЎрЃЮрЃЊрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃўрЃА рЃцрЃарЃЮрЃюрЃбрЃћрЃюрЃЊрЃќрЃћ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃбрЃљрЃюрЃљрЃЊ:

| Shortcode | рЃљрЃдрЃгрЃћрЃарЃљ | рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃў |
|-----------|--------|----------|
| `[invoice_generator]` | рЃљрЃ«рЃљрЃџрЃў рЃўрЃюрЃЋрЃЮрЃўрЃАрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃцрЃЮрЃарЃЏрЃљ. | рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃћрЃЉрЃБрЃџрЃў рЃърЃћрЃарЃАрЃЮрЃюрЃљрЃџрЃў |
| `[invoice_accountant_dashboard]` | рЃЉрЃБрЃдрЃљрЃџрЃбрЃарЃўрЃА рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃЊрЃљрЃцрЃљ (View/Edit рЃБрЃцрЃџрЃћрЃЉрЃћрЃЉрЃўрЃЌ). | рЃЉрЃБрЃдрЃљрЃџрЃбрЃарЃћрЃЉрЃў/рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў |

## ­ЪЊі рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ

рЃърЃџрЃљрЃњрЃўрЃюрЃў рЃЦрЃЏрЃюрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃфрЃ«рЃарЃўрЃџрЃћрЃЉрЃА:
* `cig_invoices` - рЃўрЃюрЃЋрЃЮрЃўрЃАрЃўрЃА рЃ░рЃћрЃЊрЃћрЃарЃў (рЃюрЃЮрЃЏрЃћрЃарЃў, рЃАрЃбрЃљрЃбрЃБрЃАрЃў, рЃЌрЃљрЃарЃўрЃдрЃћрЃЉрЃў).
* `cig_invoice_items` - рЃўрЃюрЃЋрЃЮрЃўрЃАрЃўрЃА рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃў (JSON-рЃўрЃА рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ рЃарЃћрЃџрЃљрЃфрЃўрЃБрЃарЃў рЃЕрЃљрЃгрЃћрЃарЃљ).
* `cig_payments` - рЃњрЃљрЃЊрЃљрЃ«рЃЊрЃћрЃЉрЃўрЃА рЃўрЃАрЃбрЃЮрЃарЃўрЃљ.
* `cig_customers` - рЃЎрЃџрЃўрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЉрЃљрЃќрЃљ.
